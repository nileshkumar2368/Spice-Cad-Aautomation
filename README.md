# SPICE CAD Automation Tool

## Overview
This project automates post-processing of HSPICE/NgSPICE simulation outputs to extract
key metrics such as power, delay, and I–V characteristics.

## Features
- DC I–V extraction
- Transient delay calculation
- Batch-friendly automation
- CSV reports and plots
- CAD-style workflow

## Tech Stack
Python, Pandas, NumPy, Matplotlib, HSPICE, NgSPICE

## How to Run
```bash
pip install -r requirements.txt
python main.py
